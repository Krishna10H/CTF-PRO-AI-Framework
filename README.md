# CTF-PRO-AI Framework
**Created by Dasa Krishna Chaitanya**

A modular automation framework for CTF challenges and Penetration Testing.

## Features
- **Auto-Recon:** Nmap service discovery.
- **Web Pwn:** Directory busting & API discovery.
- **Crypto Pwn:** Automated ROT13, Base64, and Hex decoding.
- **Forensics:** File hashing and Image EXIF metadata extraction.
- **Exploit DB:** Searchsploit integration.
